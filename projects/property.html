PropertyPro-Lite


This project was created using javascript, PERN[Postgres, Express, React, Nodejs]

  • Users can sign up and sign in.
  • Users can post a property advert.
  • Users can update the details of a property advert.
  • Users can get(view) a single/all property adverts.
  • Users can delete a property advert.
  • Users can mark a property advert as sold.
  • Users can get specific types of a property.
  • Users can flag a posted property AD as fraudulent.

To test this code: get started.
Clone the github repo using: `https://github.com/funmi5/PropertyPro-Lite.git`
After cloning, cd in the repo and run npm install.
Run `npm run start:dev to start the server.`

The present API routes are:
Using this heroku link the project was hosted on - `https://propertypro-lite7.herokuapp.com`
POST /api/v1/auth/signup - the user can sign up.
POST /api/v1/auth/signin - the user can sign in.
POST /api/v1/post-property - the user can post a new property AD.
PATCH /api/v1/update-property/:id - the user can post a new property AD.
PATCH /api/v1/mark-property/:id/sold - the user can mark the property AD as sold.
DELETE /api/v1/delete-property/:id - the user can delete a property AD.
GET /api/v1/all-properties - the user can get all property ADs.
GET /api/v1/one-property/:id - the user can get a single property AD.
GET /api/v1/property/type/:id - the user can get all specific types of property ADs.
POST /api/v1/property/flag/:id - the user can flag a fraudulent property AD.
